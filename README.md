#  Práctica del taller: API REST con Node.js y Express

Este es un servidor web básico construido con Node.js y Express. Proporciona una API RESTful sencilla para gestionar una lista de usuarios (CRUD básico) y rutas de prueba.

## Requisitos previos

Antes de comenzar, asegúrate de tener instalado:
* Node js

## Instalación

  1.  Descarga o navega a la carpeta del proyecto.
  2.  Abre tu terminal en el directorio del proyecto.
  3.  Instala las dependencias ejecutando:

```bash
npm install
